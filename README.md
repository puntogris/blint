# blint
Open source stock & inventory manager for Android.

Used in this project:</br>
  * Kotlin
  * MVVM
  * Hilt
  * Room
  * Camera X with QR code analizer.
  * Firebase Authentication
  * Firebase Firestore
  * Firebase Functions
  * Firebase Storage
  * Vuepress
